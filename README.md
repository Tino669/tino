# Logiciel de Gestion de Stock 📦

Cette application gratuite permet aux petits commerçants de gérer facilement leurs produits, leurs entrées/sorties de stock, et de visualiser des rapports simples.

## Fonctionnalités
- 📥 Entrée et sortie des produits
- 📊 Rapport de stock
- 🔐 Gestion des utilisateurs

## Technologies utilisées
- Frontend: HTML / CSS / JavaScript
- Backend: Node.js
- Base de données: MongoDB
